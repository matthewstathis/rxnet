Medicare Part B (Red, White & Blue Card)

  • Condor code: /22345
  • Diagnosis code: Z23 (Encounter for immunization)

Billable items:

  • Blood glucose meter, strips, lancets (must bill meter first sometimes)
  • CGMs — must bill reader before sensors
  • Nebulizer solution (not devices)
  • Anti-organ transplant rejection drugs
  • Insulin for insulin pumps

Running eligibility with patient’s SSN may return Med A/B ID.

Some Medicare Part D plans show the Med B ID when processing test strips/lancets and viewing the TP popup.

Paxcess Patient Assistance Program

Only for: Medicare, Medicaid, Uninsured
Only while AssistRx cannot process TPPC 41395 claims.

What pharmacy teams need to know

  1. Always bill patient’s primary insurance first.
  2. If not covered or high copay → explain Paxcess program.
  3. Patients can enroll online or by phone.
  4. Bill Paxcess PAP using the provided ID after enrollment.

If AssistRx is unable to process:
Reject code 92 – Host Processing Error
Message: *CVS* PHARMACY MUST call PAXCESS

Obtain patient consent and mailing address → call Paxcess → prescription is mailed to patient at $0.
